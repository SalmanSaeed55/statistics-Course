# statistics-Course

This repository documents my self-study journey through Mike X Cohen's Udemy course, **Master statistics & machine learning: intuition, math, code**.

## Project structure

| Path | What it contains | Purpose |
| --- | --- | --- |
| `01_Section_Summaries/` | Section-by-section summary notes | My own written understanding of each course section. |
| `02_Course_Material/` | Instructor-provided materials and notebooks | Hands-on practice using the course content provided by the instructor. |


## Course structure file (Excel)

`statsML_video_overviews.xlsx` is an Excel-friendly map of the course structure, used to track sections/videos and quickly navigate what to study next.

## How I use this project to learn statistics

I use a repeatable cycle tied to each project area:

1. Use `statsML_video_overviews.xlsx` to get an overview of the next section/video in the course.
2. Study and run the instructor material in `02_Course_Material` for that section.
3. Write or refine my summary in `01_Section_Summaries` to explain the section in my own words.
4. Revisit summaries and notebooks for spaced review and deeper understanding.

## Learning goal

Build strong statistical intuition, connect theory to practical coding examples, and keep a personal reference I can review anytime.
